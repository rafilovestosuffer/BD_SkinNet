# BD_SkinNet
BD-SkinNet: Diffusion-Augmented Multi-Scale CBAM-Swin Transformer for Bangladeshi Clinical Skin Disease Classification

## Dataset

This project uses the **SkinDiseaseBD** publicly available dataset.

- **Source:** [SkinDiseaseBD: A Dataset of Common Skin Disease Images of Bangladesh](https://doi.org/10.17632/9ggd3shdr7.2)
- **Platform:** Mendeley Data
- **License:** CC BY-NC 4.0
- **Classes:** Dermatitis, Eczema, Scabies, Tinea Ringworm, Vitiligo (1,612 images, 512×512px)
- **Citation:** Islam, S., Hossain, S., Ahmed, M. R., & Ibrahim, M. (2026). *SkinDiseaseBD: A Dataset of Common Skin Disease Images of Bangladesh*. Mendeley Data. DOI: 10.17632/9ggd3shdr7.2

---

This project also uses the **SkinDisNet** publicly available dataset.

- **Source:** [SkinDisNet: A Multi-Class Clinical Images and Metadata for Skin Disease](https://doi.org/10.17632/yj3md44hxg.2)
- **Platform:** Mendeley Data
- **License:** CC BY-NC 4.0
- **Classes:** Atopic Dermatitis, Contact Dermatitis, Eczema, Scabies, Seborrheic Dermatitis, Tinea Corporis (1,710 images + 11,970 augmented, 512×512px)
- **Citation:** Sultana, M., Sajib, R. H., Badhan, I. A., Ahmed, M., & Bushra, M. Y. (2025). *SkinDisNet: A Multi-Class Clinical Images and Metadata for Skin Disease*. Mendeley Data. DOI: 10.17632/yj3md44hxg.2
